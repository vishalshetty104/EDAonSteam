Exploratory Data Analysis on Steam

1. Requirements - Google Drive, Google Collaboratory, Tableau Reader

2. Steps to run the project:
	
   FOR DATA CLEANING/PROCESSING code

	- Upload .ipynb file, .csv files and .ipynb_checkpoints folder to google drive
	- Open .ipynb file using google collaboratory (DataCleaning.ipynb should be opened first)
	- Press connect on top right 
	- run each cell one-by-one

   FOR DATA VISUALIZATION 
	
	- Open .twbx file using Tableau Reader
